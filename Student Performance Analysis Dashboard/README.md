# 📊 Student Performance Analysis Dashboard | Power BI

## 📌 Project Overview

The Student Performance Analysis Dashboard is an interactive Power BI project designed to analyze students' academic performance across Mathematics, Reading, and Writing scores. The dashboard provides insights into how factors such as gender, ethnicity, parental education, and test preparation influence student performance.

This project demonstrates data visualization, dashboard development, KPI creation, and business intelligence skills using Power BI.

---

## 🎯 Objectives

- Analyze overall student academic performance.
- Compare average scores across different subjects.
- Identify performance differences by gender and ethnicity.
- Evaluate the impact of parental education on student scores.
- Measure the effect of test preparation on academic achievement.
- Highlight top-performing students.

## 📈 Key Insights

### Overall Performance

- The overall average student score is **61.18%**.
- Mathematics has the highest average score among the three subjects.

### Gender Analysis

- Male and female students show relatively similar academic performance with only minor differences.

### Ethnicity Analysis

- Students from **Group A** achieved the highest average score percentage.
- Performance across ethnic groups remains fairly balanced.

### Parental Education

- Students whose parents have higher education levels generally perform slightly better.
- Academic performance shows a positive relationship with parental education.

### Test Preparation

- Students who completed the test preparation course scored higher on average than those who did not.

### Top Performers

- The dashboard highlights the top five students based on combined Math, Reading, and Writing scores.
